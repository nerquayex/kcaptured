## 🚀 Quick Launch (Copy & Paste)

**3 files you need to know about:**

### 1. Add Your Logo
```
Create: /public/images/logo.png (180×40 pixels)
Format: PNG with transparent background
Size: < 50KB
```

### 2. Add Your Favicon
```
Create: /public/favicon.ico (64×64 pixels)
Format: ICO or PNG
Size: < 20KB
Tip: Can be letter "K" or camera icon
```

### 3. Get Formspree ID
```
Go to: https://formspree.io
Create: New form
Copy: Form ID (looks like: f/abc123xyz)
Edit: components/contact-form.tsx line 75
Replace: YOUR_FORM_ID with your actual ID
```

**Then deploy to Vercel and you're done!**
